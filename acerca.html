Acerca del proyecto

Este es un proyecto creado para la asignatura de: TECNOLOGIAS PARA DESARROLLO DE APLICACIONES WEB, el objetivo es usar el API del SMN de México para mostrar información acerca del clima. Son 3 vistas principales, una vista informatíva, una vista donde se muestren los climas de todos los municipios de cada estado del país y una vista donde se pueden filtrar estos datos para saber el clima de algun municipio o estado en específico.
